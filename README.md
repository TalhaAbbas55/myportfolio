# Talha Abbas Portfolio

Personal portfolio built with Next.js, Tailwind CSS, Framer Motion, and Aceternity-style UI effects.

## Overview

This repository contains the portfolio site for Talha Abbas, a software engineer with experience across frontend, backend, mobile, and product delivery. The site highlights:

- Professional summary and contact details
- Work experience across software engineering and full-stack roles
- Selected projects
- Core technical capabilities
- Education, achievements, and interests

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js based visual components

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site locally.
